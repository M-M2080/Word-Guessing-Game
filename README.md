# Word Guessing Game (WGG)

A simple word guessing game written in Python.

## Description
In this game, the program randomly selects a word and the player must guess it letter by letter.
The number of allowed guesses depends on the length of the chosen word.
Players can also provide their own custom word list.

## Features
- Random word selection
- Custom word list support
- Guess limit based on word length
- Win / Lose tracking

## How to Run
Make sure Python is installed, then run:

```bash
python main.py
